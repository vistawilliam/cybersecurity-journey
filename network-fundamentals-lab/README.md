# Network Fundamentals Lab

## Objective

Learn the networking concepts used daily by cybersecurity analysts.

## Topics Covered

### Ping

Used to test connectivity between devices.

Example:

```bash
ping google.com
```

### DNS

Used to translate domain names into IP addresses.

Example:

```bash
nslookup google.com
```

### Traceroute

Shows the path packets take through the network.

Example:

```bash
tracert google.com
```

### TCP vs UDP

TCP:
- Reliable
- Connection-oriented

UDP:
- Faster
- Connectionless

### Ports

Examples:

- 22 SSH
- 80 HTTP
- 443 HTTPS
- 53 DNS

## Tools Used

- Windows Command Prompt
- Linux Terminal
- Nmap
